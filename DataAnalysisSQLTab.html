Exploratory Data Analysis using SQL and Tableau

 Exploratory Data Analysis using SQL and Tableau

COVID-19 Data Analysis and Visualization

Project Overview: Created a SQL database to analyze COVID-19 data, focusing on COVID deaths and vaccination rates worldwide. Utilized advanced SQL techniques including Joins, CTEs, Temp Tables, Window Functions, Aggregate Functions, and Data Type Conversions to explore global data on confirmed COVID-19 deaths. Built a dashboard in Tableau to report findings from the COVID-19 exploratory analysis.

Objectives

  • Database Creation: Build a comprehensive SQL database to store and analyze COVID-19 data.
  • Data Analysis: Utilize SQL techniques to perform in-depth analysis of COVID deaths and vaccination rates.
  • Data Visualization: Create informative visualizations using Tableau to report findings from the analysis.

Techniques Used

  • SQL Joins: Combined multiple tables to integrate COVID-19 death and vaccination data.
  • CTEs and Temp Tables: Organized and preprocessed data for analysis.
  • Window Functions: Applied to calculate moving averages, rankings, and other windowed aggregations.
  • Aggregate Functions: Used to compute summary statistics and metrics.
  • Data Type Conversions: Converted data types for compatibility and analysis.
  • Views: Created views to encapsulate complex SQL queries for easier access and reuse.

Process and Implementation

Established a SQL database structure to store COVID-19 data efficiently, integrated data from multiple sources, performed exploratory analysis using advanced SQL techniques, and built a dashboard in Tableau to visually represent key findings from the analysis.

Results and Impact

The project provided comprehensive insights into global trends in COVID deaths and vaccination rates, enabling data-driven decision-making in public health policy and intervention strategies.
